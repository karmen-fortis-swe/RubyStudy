### Running Ruby code:
- to run a Ruby file:
```
ruby "name_of_your_file"  
```

- define a method:
```
def my_method(myinfo)
    puts myinfo
end
mymethod "my info goes here"
```
- "irb" stands for interactive Ruby. Can be used to create a Ruby environment in the terminal.
